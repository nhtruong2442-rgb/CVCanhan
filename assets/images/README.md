# Image Assets - Instructions

## How to Add Your Images

Place your images in this folder (`assets/images/`).

### Required Images:

| Image File | Description | Recommended Size |
|---|---|---|
| `profile.jpg` | Your professional headshot / profile photo | 500x500px minimum |
| `about.jpg` | Photo at construction site or professional setting | 600x800px minimum |
| `project-1.jpg` | High-rise building project photo | 800x600px minimum |
| `project-2.jpg` | Industrial factory project photo | 800x600px minimum |
| `project-3.jpg` | Infrastructure project photo | 800x600px minimum |
| `project-4.jpg` | Commercial building project photo | 800x600px minimum |
| `project-5.jpg` | Residential complex project photo | 800x600px minimum |

### Notes:
- The website will show elegant placeholder icons if images are not found.
- Use `.jpg`, `.jpeg`, or `.png` format.
- Optimize images for web (compress before uploading).
- Recommended free tool: [TinyPNG](https://tinypng.com/)

### Quick Replacement:
1. Name your photo exactly as listed above
2. Drop it into this `assets/images/` folder
3. The website will automatically display your image
